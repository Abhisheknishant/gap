Description of fields

Component

 The part of GAP this is for.

Contact

 The person who proposed the problem, respectively someone who is interested in having this problem resolved. (This is not necessarily the authority on the problem.)

Time

 How long will it take:
1Under a day
2A few days
3A few weeks
4Several Months
5Over a year

Need

 How desirable is this addition (1=someone mentioned it, 5=extremely desirable)

Math

 Level of mathematical knowledge required
1Middle School
2Basic abstract algebra
3Graduate level algebra
4More specialized knowledge of the particular topic
5Requires new research

Program

 Level of programming skills required
1Text editing
2Basic GAP commands
3
4Fluency in GAP
5Expert knowledge of GAP or C

Description

 A description of the problem arising or the task to be done.

Usage

 A potential manual page for the new command

References

 References to relevant literature, for example a published form of the article.

Remarks

 Remarks (probably by other persons)
