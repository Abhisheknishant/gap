GAP Character Table Library

Character Tables in GAP

This overview contains the following views to present information about the contents of the GAP Character Table Library.

All GAP Character Table Names by Group Order
This view lists the names of all character tables that are available in the GAP Character Table Library, sorted according to their group orders.
Simple Groups by Group Order
This view lists the finite simple groups up to order 109 and their orders. Groups whose character tables are available in the GAP Character Table Library appear as links.
