Some GAP HOWTOs

The aim of these web pages is to document some internal procedures we agreed on or which have developed over time. It is intended as an aid to developers, not as a guarantee to the outside or package authors that these procedures will always be followed in eternity. (Procedures that should be documented to package authors should be documented in one of the manuals.)

It is compiled from various other files in the CVS directories. These files have not been deleted but should become obsolete soon. Everything to which this starting page links is under dev/howtos.

These files are written in HTML to permit reasonable display, but there is otherwise nothing special about the format. Display is simply by 

netscape "gap4/dev/HOWTO.htm"
in the local gap4 directory.

Additions or corrections are welcome.
